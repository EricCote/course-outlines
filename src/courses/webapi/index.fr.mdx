## Description

ASP.NET Core permet de créer des services REST, aussi appelés services
Web API. Cela permet de créer des infrastructures pour que nos
différents systèmes puissent échanger de l'information. Les Web
API sont les blocs sur lesquels on pourra bâtir des architectures SOA,
micro-services et nano-services. Les données sont transférées dans un
format JSON.

Cette formation utilise la dernière version de .NET. (Version 6 au moment d'écrire ces lignes)
