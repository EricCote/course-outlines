## Description

ASP.NET 5 remplace la plateforme MVC provenant du vieux .NET Framework 4.x. C'est la nouvelle plateforme basée sur .NET Core.

Apprenez à migrer vos vieilles applications MVC (2010-2019) vers .NET 5.
