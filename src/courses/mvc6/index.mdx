## Description (TODO)

ASP.NET 6 est la nouvelle version recommandée pour le développement d'applications web côté serveur. Le détail important, l'architecture recommandée est MVC, un patron de développement basé sur un découpage composé de Modèles, de Vues et de Contrôleurs.

Venez bâtir plusieurs petites applications web, qui vous permettront de comprendre les meilleures pratiques et usages.
